# Food Ordering Website

![Project Logo](/FoodyZone.svg)

A dynamic Food Ordering website built with React and styled with CSS. The website fetches data from an API, allowing users to filter and view different food items based on their preferences.


## Features
- **Dynamic Rendering:** The website dynamically fetches and displays food items from an API.
- **Filter Functionality:** Users can filter food items based on different categories (e.g., All, Breakfast, Lunch, Dinner).
- **Search Functionality:** A search bar allows users to search for specific food items.
- **Responsive Design:** The website is designed to be responsive, providing a seamless experience across devices.
